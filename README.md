# IceWM-haikudark

Dark IceWM theme based on [Haiku](https://www.haiku-os.org/).

# Screenshot

![](https://i.ibb.co/N6dq95XD/2026-05-31-135924-1366x768-scrot.png)

# Installation
Download this repo (or just the `.7z` file). Decompress `HaikuDarkMod.7z`. Move the `HaikuDarkMod` folder into `~/.icewm/themes/`

# Important

**DISCLAIMER:** Super buggy and ugly, didn't put much effort into this one, uploading it just in case someone is interested, because the only dark Haiku IceWM theme I could find was basically the one we already know, but inverted lol.

- [IceWM Haiku theme](https://www.gnome-look.org/p/1018245) - Original theme, I just modified it using new assests.
- [Haiku OS Themes (Mohawke_Dark)](https://www.darkartistry.com/haiku-os-themes/) - Copied most of the design from here.
- [ReVista (Dark)](https://github.com/x35gaming/ReVista) - GTK theme used in the screenshot.
- [Haiku OS iconset](https://www.gnome-look.org/p/2087825) - Haiku icons.
